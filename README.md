<h1 align="center"> Caesar </h1>

<p align="center">echo 框架构建的简单密码管理 Web 应用后端 API</p>

### 环境需求
* go 版本不低于 1.18
* Mysql 版本不低于 5.7
* Redis 版本不低于 5.0

### 安装使用
* 下载安装
  ```shell
    git clone git@github.com:uuk020/caesar.git && go build main.go 
  ```
* 初始化
  ```text
  把 settings-dev.yaml 修改为 settings.yaml
  ```
* 编译 go
  ```shell
  go build main.go
  ```
